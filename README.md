# APU Medical Centre (AMC) Management System

This project is developed as part of the Object-Oriented Programming coursework at Asia Pacific University (APU).

The system is designed to streamline medical centre operations by providing functionalities for Managers, Staff, Doctors, and Customers. It supports user management, appointment booking, doctor assignment, payment processing, feedback management, appointment history tracking, and report generation.

## Technologies Used
- Java 21
- Java Swing (GUI)
- Object-Oriented Programming (OOP)
- Text File Storage
- IntelliJ IDEA

## Key Features
- User Authentication
- User Management
- Appointment Booking
- Doctor Assignment
- Payment Processing
- Receipt Generation
- Feedback and Comments
- Appointment History
- Analytical Reports
